# Junta Directiva

## Participantes
- Christian (Product Manager)
- Luis (Arquitecto)
- Pablo (Cliente)
- Angel (Auditor)
- Sebas (Scrum Master)
- Brayan (Desarrollador)

## Tema
Mejoras para la librería/tienda, enfocadas en organización, atención al cliente y uso de herramientas para optimizar el servicio.

## Desarrollo
Durante la reunión se discutieron varias ideas para mejorar el funcionamiento del negocio.
Sebas propuso el uso de herramientas de inteligencia artificial para responder mensajes de clientes por WhatsApp de forma más rápida y ordenada.
Christian mencionó que es importante que las soluciones se implementen en el menor tiempo posible, priorizando rapidez en la ejecución.
Luis habló sobre la necesidad de mejorar la organización del personal, asignando mejor las tareas para evitar desorden y pérdida de tiempo.
Angel propuso implementar un sistema para cobrar y llevar control de los productos, con el fin de tener mayor orden y evitar errores.
Brayan sugirió la creación de una página web para mostrar productos, mejorar la comunicación con clientes y apoyar en la publicidad del negocio.
Pablo, como cliente, comentó la importancia de recibir un buen servicio, tiempos rápidos y que siempre haya disponibilidad de productos.
También se discutió la posibilidad de implementar servicio a domicilio (delivery), mejorar la publicidad del negocio y establecer tiempos más claros de atención al cliente.

Además, se consideró separar la librería de la tienda en dos locales diferentes, pero se concluyó que requería muchos recursos y no era viable en este momento.
Finalmente, se habló sobre mejorar el control de inventario y la organización del espacio para evitar errores y mejorar la atención.
