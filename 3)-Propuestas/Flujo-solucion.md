# Flujo de la solución

## Descripción
Se propone mejorar el funcionamiento de la librería/tienda mediante un proceso más ordenado para atender a los clientes, controlar productos y reducir errores.

El sistema se enfoca en:
- Revisar si hay productos disponibles antes de vender
- Llevar un control básico de lo que falta
- Mejorar la atención con apoyo de herramientas como WhatsApp
- Evitar errores en impresiones

## Explicación
El flujo inicia cuando un cliente solicita un producto o una impresión.

Primero, se verifica si el producto está disponible.  
Si está disponible, se atiende al cliente normalmente.

Si no está disponible, se revisa una lista de control para ver si se puede resolver con el stock existente o si es necesario programar una compra.

En el caso de impresiones, se debe revisar antes de imprimir para evitar errores.

Este proceso permite reducir pérdidas de tiempo, mejorar la atención al cliente y mantener un mejor control del negocio.
