# Proyecto 2 - Comunicación Asertiva

## Descripción

Este proyecto consiste en la simulación de un entorno colaborativo de desarrollo de software, en el cual se aplican técnicas de comunicación asertiva, negociación y análisis de requerimientos a partir de un negocio real.

El equipo realizó entrevistas a negocios, identificando problemas en su operación y proponiendo una solución tecnológica basada en un sistema digital.

---

## Objetivo

Desarrollar habilidades de:
- comunicación efectiva  
- trabajo en equipo  
- análisis de requerimientos  
- uso de herramientas colaborativas  

---

## Problema identificado 1

El negocio presenta dificultades en:
- gestión de pedidos por múltiples canales  
- control manual de inventario  
- desorganización en la información  

---

## Solución propuesta

Se propone el desarrollo de un sistema digital que:
- centraliza pedidos  
- controla inventario  
- gestiona servicios de impresión  
- mejora la atención al cliente
